## Key features

### Command palette

Access any utility instantly with the command palette. Press `⌘K` (Mac) or `Ctrl+K` (Windows/Linux) to open it, or simply press `/` from the homepage. Start typing to search across all 22 utilities.

<Tip>
You can also press any letter key from the homepage to open the command palette with that letter pre-filled.
</Tip>

### Keyboard shortcuts

TryDevUtils is built for keyboard-driven workflows:

- `⌘K` or `/` - Open command palette
- `?` - Show all keyboard shortcuts
- `Esc` - Return to homepage or close dialogs
- `⌘Enter` - Execute/format/convert in any utility
- `⌘Shift+C` - Copy output to clipboard
- `⌘L` - Clear all inputs

Press `?` anywhere in the app to see the complete list of shortcuts.

### Favorites system

Mark your most-used utilities as favorites by clicking the heart icon on any utility card. Your favorites appear at the top of the homepage for quick access.

<Note>
Favorites are stored locally in your browser using localStorage and never leave your machine.
</Note>

### Multi-platform support

Choose the platform that works best for you:

- **Web app**: Instant access from any modern browser
- **Desktop app**: Native performance with offline support for macOS, Windows, and Linux
- **Chrome extension**: Quick access from your browser toolbar

All three platforms share the same features and privacy-first approach.
